# Flow Studio

A responsive marketing site for Flow Studio, an independent graphic design practice offering brand identity, web, print, and recurring flyer design subscriptions. The interface uses a tactile press-proof visual system with registration marks, ink swatches, ticket-inspired pricing, and original CSS artwork.

## Technology

- TanStack Start and TanStack Router
- React 19 and TypeScript
- Vite with Netlify's TanStack Start adapter
- Tailwind CSS tooling with a custom global CSS design system
- Lucide React icons

## Local Development

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

The project is configured for deployment on Netlify through `netlify.toml` and `@netlify/vite-plugin-tanstack-start`.
